Quick Start Examples

1. Login and get JWT token:

curl -X POST http://localhost:8081/api/v1/auth/login \ -H "Content-Type: application/json" \ -d '{"username": "admin", "password": "admin123"}'

2. Scrape Amazon product:

curl -X POST http://localhost:8081/api/v1/amazon/scrape \ -H "Authorization: Bearer YOUR_JWT_TOKEN" \ -H "Content-Type: application/json" \ -d '{"url": "https://www.amazon.com/dp/B08N5WRWNW"}'

3. Get analytics stats:

curl -H "Authorization: Bearer YOUR_JWT_TOKEN" \ http://localhost:8081/api/v1/analytics/stats
